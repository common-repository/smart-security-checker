== Installation ==

It's easy to install. Just Install the plugin and you will see a new admin new "Smart Security Checker" click this and there will visible a button "check for security issue" Click this button there will automatically check your website's security issue and will show a notice about the result.

e.g.

1. Upload `plugin-directory` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Just install and active.
4. Click admin new "Smart Security Checker"
5. Click button "check for security issue"
